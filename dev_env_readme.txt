To install the dev environment used for this plugin:
Install qgis 2.18 using the OSGeo4W network installer, installing the following packages:
    -   qgis-ltr
    -   qgis-ltr-common
    -   qt4-devel
    -   pyqt4

Set the following environment variables in an appropriate way, change OSGeo4W location if needed:
    PATH variable:
    -   C:\OSGeo4W64\bin
    -   C:\OSGeo4W64\apps\qgis-ltr\bin
    
    PYTHONPATH variable:
    -   C:\OSGeo4W64\apps\qgis-ltr\python
    -   C:\OSGeo4W64\apps\Python27
    -   C:\OSGeo4W64\apps\Python27\Lib\site-packages
